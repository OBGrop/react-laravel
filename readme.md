

## Getting Started

Clone the project repository by running the command below if you use SSH

```bash
git clone https://github.com/OBGrop/react-laravel.git
```

After cloning, run:

```bash
composer install
```

```bash
npm install
```
start the application:
```bash
php artisan serve
```
